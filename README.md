# Scribewise

Private, on-device transcription for macOS. Record your mic and system audio,
get an accurate transcript, and a clean summary. Nothing leaves your Mac.

**[Download the latest version](https://github.com/nicblunck/scribewise/releases/latest/download/Scribewise.zip)**

Requires macOS 15 (Sequoia) or later on Apple Silicon.

---

This is the public home for Scribewise downloads and the website. The app's
source code lives in a separate private repository. Each release here contains:

- `Scribewise.zip` — the notarized app
- `appcast.xml` — the auto-update feed the app reads

The website is served from `index.html` via GitHub Pages.
